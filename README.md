# Memorization_Perturbation
Large Language Models (LLMs) demonstrate profound capabilities in generating fluent text, yet distinguishing genuine abstract generalization from verbatim retention of training data remains a critical challenge for privacy and model safety. Current Membership Inference Attacks (MIAs) predominantly evaluate verbatim sequences against global probability distributions, overlooking the structural topology of retained data. In this paper, we introduce a dynamic evaluation framework that maps the local semantic manifold of memorized sequences via controlled, meaning-preserving syntactic perturbations and exposes vulnerabilities in MIA evaluation techniques.
